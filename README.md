# TO-DO-LIST-APP
# Problem Statement
Managing daily tasks manually often leads to missed deadlines, poor organization, and reduced productivity. Users need a simple, digital solution that helps them add, track, and update tasks efficiently. A lightweight and intuitive To-Do List application can simplify task management, improve focus, and enhance productivity by providing a structured way to organize daily activities.
# Objectives
To design a simple and user-friendly application for managing personal tasks.
To implement CRUD functionality: create, view, update, and delete tasks.
To allow users to set priorities and due dates for better task planning.
To store tasks persistently using a database (SQLite).
To build a responsive and clean interface using HTML, CSS, and Flask templates.
To ensure smooth navigation through filtering (All, Active, Completed) and sorting options.
# Features
Add new tasks with title, description, due date, and priority.
Edit and update existing tasks.
Mark tasks as completed or active.
Delete tasks permanently.
Filter tasks: All / Active / Completed.
Sort tasks by created time, due date, or priority level.
Clean UI with responsive design.
Persistent storage using SQLite.
Flash messages for user feedback (task added, updated, deleted, etc.).
